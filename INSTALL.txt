REQUIREMENTS:
=============
   Needs :
     - ROOT (https://root.cern.ch/)
     - CMAKE
     - Doxygen
     - Graphviz

INSTALL:
=============
	
  * In the GRPCAnalysor root directory, do:

    mkdir build
    cd build
		cmake ..
    make doc
    make intall 
  Enjoy !  :)
